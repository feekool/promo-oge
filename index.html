📦 Установка форума для ОГЭ за 1 минуту

  1. 1
    Создайте аккаунт на GitHub Бесплатная регистрация на github.com (можно через школу)
  2. 2
    Откройте шаблон ОГЭ
    github.com/feekool/feekool.github.io
  3. 3
    Нажмите "Use this template" → "Create a new repository". Назовите: "oge-math-2026" или "oge-russian-course"
  4. 4
    Включите GitHub Pages: Settings → Pages → ветка main → Save. Через 60 секунд форум онлайн!
  5. 5
    Настройте под свой предмет 🎯 Замените темы, загрузите задания ОГЭ, добавьте расписание консультаций.
💡 Для школ и центров: Создайте отдельные форумы для каждого предмета или группы учеников. Все форумы управляются из одного аккаунта.
